# Demo

The demo runs on docker compose; the infrastructure is defined in [`compose.yaml`](compose.yaml) and [`infra`](infra).

There are multiple independent demo apps residing in directories like [`table-api-python`](table-api-python).

The demo apps are also used for integration tests.
